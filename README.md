# Warthunder-External

Optimized external ESP framework for War Thunder.

## Important

This source does **NOT** include a kernel driver or driver communication implementation.

The original driver communication layer has been intentionally removed from the repository.

If you plan to use this project with anti-cheat protected environments such as BattleEye, you must implement your own communication layer inside:

```cpp
usermode/impl/memory/
```
